Adicionar, Editar, Deletar Páginas do Site

Nota: Para usar essa ferramenta, você deve ter direitos de administrador.

Para adicionar, alterar ou remover páginas do site, modificar, siga os passos:

  1. No menubar em seu Meu Workspace, clique em Sites. Uma lista de worksites será exibida.

  2. Nessa lista, clique no Site Id do worksite que você deseja alterar.

  3. No formulário que aparece, clique no botão Páginas (localizado na parte inferior do formulário).

  4. Você verá uma lista de páginas definidas no worksite. Siga os passos abaixo:

Adicionar uma página

Para adicionar uma página, no topo do formulário, clique em Nova Página. Complete os campos que aparecem no formulário. Quando você terminar, clique em Salvar.

Alterar uma página

Para alterar as informações de uma página, na lista de páginas, clique no Page Id da página que você deseja alterar. Faça as alterações no formulário que aparece. Quando terminar, clique em Salvar.

Remover uma página

Para remover uma página, na lista de páginas, clique no Page Id da página que você deseja remover. No topo do formulário que aparecer, clique em Remover Página.
