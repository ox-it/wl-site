Ferramenta Sites: Resumo

O que faz

Nota: Para usar essa ferramenta, você deve ter direitos de administrador.

Com a ferramenta Sites, administradores do sistema podem criar novos sites e modificar e remover os já existentes. Os usuários com direitos de administrador verá a ferramenta de sites no Meu Sitemenubar.

Ao criar ou editar um site, o administrador pode selecionar quais ferramentas serão exibidas na tela principal da página inicial do site. O administrador também pode renomear a página inicial e criar páginas iniciais adicionais para o site.

Principais conceitos

Atribuindo papéis: A ferramenta Sites permite ao administrador especificar qual papel será atribuido por padrão, para pessoas que se inscrevem no site. O proprietário do site pode mais tarde mudar o papel de um participante com a ferramenta Site Info.

Adicionando ferramentas: Quando um administrador adiciona ferramentas para uma página inicial em particular, essas ferramentas aparecem na tela principal, não no menu.

Conderações

  • Quando o aplicativo é instalado pela primeira vez, o administrador é o único com a conta pré-instalada. O administrador pode, então, criar sites.

  • O modelo de site é o site com o siteID !user. Administradores podem editar este site para incluir ou excluir quaisquer ferramentas que deseja exibir para seus usuários do Meus Workspace.
