Adicionar, rever, ou deletar um site

Nota: Para usar essa ferramenta, você deve ter direitos de administrador.

Nessa página:

  • Adicionar um site
  • Rever um site
  • Deletar um site

Adicionar um site

Para adicionar um site, siga os passos:

  1. No menubar em Meu Workspace, clique em Sites.

  2. Clicar em Novo Site.

  3. Preencha o formulário que aparece. Campos obrigatórios são marcados com um asterisco vermelho. Para mais informações sobre como adicionar páginas a um site, veja Adicionar/Editar/Deletar Páginas do Site.

  4. Quando você terminar de digitar as informações do site, clique em Salvar.

Rever um site

Para rever um site, siga os passos:

  1. No menubar em Meu Workspace, clique em Sites.

  2. A partir da lista de sites que aparece, clique no Site Id do site que você deseja rever.

  3. Faça as alterações desejadas no formulário que aparece e, em seguida, clique em Salvar. Para mais informações sobre como adicionar páginas a um site, veja Adicionar/Editar/Deletar Páginas do Site.

Deletar um site

Para deletar um site, siga os passos:

  1. No menubar em Meu Workspace, clique em Sites.

  2. A partir da lista de sites que aparece, clique no Site Id do site que você deseja deletar.

  3. Clique em Remover Site. Quando solicitada a confirmação para esta ação, clique em Remover.
